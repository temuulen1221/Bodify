## 1. Problem Overview 
Prepare housing data for ML by:  
1. Fixing 591 missing tax values  
2. Creating "Age" from construction years  
3. Scaling lot sizes (997-62,557 → 0-1)  
4. Splitting data (60-20-20)  

---

## 2. Key Code & Results  

### Tax Fix  
```python
df['TAX'] = df['TAX'].fillna(4328)  # Mean value
```
**→ Result:** 0 missing taxes remaining  

### Age Calculation  
```python
df['AGE'] = 2025 - df['YR BUILT']  
```
**→ Output:**  
Oldest: 185 yrs │ Newest: 5 yrs │ Avg: 47 yrs  

### Data Split  
```python
train, temp = train_test_split(df, test_size=0.4)
val, test = train_test_split(temp, test_size=0.5)
```
**Final Counts:**  
- Train: 3,548

- Test: 1,182

## 3. Key Learnings   
1. **Simple Imputation Works**  
   - Quick fix but reduces variance  
   - Better: Investigate why taxes were missing  

2. **Feature Scaling Matters**  
   - Made lot sizes comparable to rooms/baths  
   - Different models need different scaling  

3. **Reproducibility**  
   - `random_state=42` ensures same splits
